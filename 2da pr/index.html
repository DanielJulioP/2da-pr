¿Cual es la funcion principal de la pagina?

Esta página web tiene como propósito principal facilitar la búsqueda de los problemas y estado de los computadores, de esta manera se identificará de manera rápida si el computador tiene alguna tipo de problema de hardware o software (tales como no estar conectado al internet,tener algun tipo de daño en los componentes del computador, no funcionarle algún tipo de aplicación como el navegador Google o no tener cierto programas importantes para el uso que se le dara,etc...), también se informara si se realizo mantenimiento y cuando fue el dia en el que se realizo Ademas de que el usuario tenga la posibilidad de agregar informacion para facilitar el arreglo del equipo.

Ventajas

Mayor facilidad de ubicar los computadores con problemas.

Poder informar si se presente un problema con el software de un equipo.

El usuario puede aportar datos de manera opcional para saber cual es el problema del equipo.

Se puede informar si presenta algun problema de hardware
